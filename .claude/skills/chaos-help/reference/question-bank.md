# CHAOS Help Question Bank

Use these only when needed. Prefer direct help if the user's intent is clear.

## Multiple active changes

```text
I found multiple active OpenSpec changes. Which one should I inspect for the next-command recommendation?
```

## README target

```text
Where should I write the workflow README?
1. `.chaos/README.md` recommended
2. `docs/chaos-workflow.md`
3. custom path
```

## Root README target

```text
You selected root `README.md`, which is project-facing. I can generate a patch preview, but I should not overwrite it silently. Proceed with preview?
```

## Weak evidence

```text
I can recommend a next command, but confidence is low because the workspace artifacts are incomplete or stale. Run `chaos:status` first?
```


## README already up to date

```text
The workflow README is already up to date. No write is needed.
```

## README outdated

```text
The workflow README is outdated. I can show the patch preview and update it with `--write` or your explicit confirmation.
```

## Non-generated README target

```text
The selected README target does not appear to be generated by `chaos:help`. I should not overwrite it unless you explicitly confirm replacement after seeing the preview.
```
