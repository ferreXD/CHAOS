## Team collaboration model (v0)

Artifacts for a change live under `.chaos/changes/<change-id>/` (per-change layout with a
`lifecycle.md` manifest). Contributors run `chaos:sync --change <change-id>` (contributor-
safe); repo-wide `chaos:sync --all` requires maintainer/repo-owner confirmation. After a
change merges into `main`, run a mainline sync. Canonical contract:
[`.chaos/changes/README.md`](../changes/README.md).
